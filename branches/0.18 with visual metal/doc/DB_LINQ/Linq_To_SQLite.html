DbLinq Project: Linq Provider for SQLite

Getting Started:

1. Download general DbLinq required items, listed on the main project page
2. Download SQLite database browser - to run the DB create script
3. Download DbLinq.SQLite.zip
4. run SQLiteMetal.exe against your existing database to create a schema .cs file
5. Add the cs file and DbLinq.SQLite.Prototype.dll to your .NET project
